# Sonnenuntergang
Sonnenuntergang (Theme #13) by silbrigthemes.


<h2>About</h2>
<p>This theme is inspired by <em>Pantone</em>&rsquo;s Color of the Year 2019, <strong>Living Coral</strong>. The design works with warm colors and a contrast between angular and round elements.</p>
<p>You can find the <a href="https://silbrigthemes.tumblr.com/post/182509061325/sonnenuntergang-theme-13-by-silbrigthemes-about" title="blog post">offical blog post here</a> and a link to the <a href="https://sonnenuntergangtheme.tumblr.com/" title="live preview">live preview here</a>.</p>

<h3>Links</h3>
<p>With the five toggles at the top of the page, you can toggle the links on and off.</p>
<p>Further below, there are three sections for each custom link: There you can the URL for the link, the link&rsquo;s name (e.g. its destination), and a custom icon (from <a href="https://honeybee.suiomi.com/" target="_blank">honeybee</a> by <em>suiomi</em>).</p>
<p>To get the icon to work, you simply have to write its name in the section, e.g. for the outlined heart icon you just write &rsquo;<strong>heart-1-o</strong>&rsquo; and you&rsquo;re done!</p>
<h3>Fonts</h3>
<p>Under &ldquo;<strong>Font</strong>&rdquo; you can select the font you want to use on your blog for texts. You can choose between <em>Open Sans</em>, <em>Roboto</em>, and <em>Source Sans Pro</em>.</p>
<p>Under &ldquo;<strong>Font Size</strong>&rdquo; you can select between three font-sizes; 16px, 14px, and 12px.</p>
<p>Under &ldquo;<strong>Title Font</strong>&rdquo; you can select between three fonts which will be used for post titles and other titles. The choices are <em>Josefin Sans</em>, <em>Kanit</em>, and <em>Oswald</em>.</p>
<p>Under &ldquo;<strong>Text Align</strong>&rdquo;, you can choose whether you want the text on your blog to be <em>left-aligned</em> or <em>justified</em>.</p>
